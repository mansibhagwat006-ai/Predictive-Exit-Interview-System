# Predictive-Exit-Interview-System
This project develops a machine learning model that predicts which employees are at a high risk of leaving the company before they actually resign. The predictions help the HR department take preventive actions to improve employee retention. 

## Project Objective
Predict employees who are at high risk of leaving the company using machine learning.

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Files
- HR_Attrition_Prediction.ipynb
- ML_Project_Charter.pdf
- requirements.txt

## How to Run
1. Install the required libraries.
2. Open the Jupyter Notebook.
3. Run all cells.
